# **App Name**: LammaFeed

## Core Features:

- User Authentication & Profiles: Secure sign-up and login via email/password, Google, or Apple. Users can create a personal profile with a photo, cover, bio, and external links.
- Scrolling Content Feed: A dynamic feed similar to TikTok/Instagram, displaying textual posts, images, and short videos (up to 60 seconds) either chronologically or based on simple 'For You' logic (followers + likes).
- Content Publishing & Engagement: Users can create and publish new text, image, or video posts. Interact with posts through liking, commenting, and reposting/retweeting functionality.
- Social Networking Features: Core social interactions including following/unfollowing other users and managing personal follower lists.
- Lamma Community Hub: Dedicated 'Lamma' section for group chat rooms centered around various topics relevant to Arabic and global communities. Users can also create public or private 'Lamma' groups.
- Multilingual UI & RTL Support: Full support for English as default and Arabic with excellent RTL (Right-To-Left) display. Language can be changed manually or automatically detected.
- AI-Assisted Content Moderation Tool: A tool to flag and review inappropriate content and detect blocked keywords across multiple languages, assisting manual moderation efforts for serious reports.

## Style Guidelines:

- Primary color: An inviting, slightly muted blue (#1E6FC9) that conveys trustworthiness and connection, contrasting well with light backgrounds.
- Background color: A very light, desaturated blue (#E7EEF7) providing a clean, open canvas for content.
- Accent color: A vibrant, refreshing greenish-cyan (#48C9E4) used for interactive elements and highlights to draw attention and add a touch of energy.
- All text: 'Inter' (sans-serif) for its modern, clear, and highly legible characteristics, suitable for both headlines and body content across different languages.
- Utilize clean, outlined icons with a consistent style that are universally recognizable and complement both LTR and RTL layouts.
- A responsive, adaptive layout that fluidly adjusts content display and navigation for seamless viewing across mobile, tablet, and desktop devices, including full RTL text and element mirroring.
- Subtle and smooth transitions for scrolling the feed, opening posts, navigating Lamma communities, and reacting to content, ensuring a fluid user experience.
